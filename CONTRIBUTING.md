# Contributing Guidelines

We welcome contributions to this project! Please follow these basic rules:

1. Fork the repository and create a new branch for your changes.  
2. Write clear, concise commit messages.  
3. Ensure your code is properly formatted and documented.  
4. Add tests where applicable and verify all tests pass.  
5. Submit a pull request with a detailed description of your changes.  
6. Be respectful and collaborative in discussions and reviews.  

Thank you for helping improve this project!
